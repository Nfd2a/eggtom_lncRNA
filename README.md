# Eggplant and Tomato lncRNA de novo annotation

Preprocessing ran with: 
- Fastp 0.23.4
- Salmon 1.10.0

Transcript assembly, merging, and classification ran with:
- Hisat2 2.1.0
- Stringtie 2.2.1
- Gffcompare 0.12.9

Filtering ran with:
- CPC2 1.0.1
- Transcoder 5.5.0
- Infernal 1.1.3
- Subread 2.1.1
